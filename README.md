# vue-course-code
Code of Vue JS 2 - The Complete Guide Course

## Installation
1. Install `Node.JS` https://nodejs.org/en/
2. `git clone https://github.com/fabriziopace/vue-course-code.git`
3. `npm install`
4. `npm run start`
